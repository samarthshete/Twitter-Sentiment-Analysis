ABOUT THE PROJECT

Fetching Live Tweets

To create a real world interface, we have created a program which shall surf twitter for quite a number of tweets based on the hashtag list prepated by the user.We have consolidated the program even more directing it to fetch tweets from a particular twiiter id.

Visualizations

In this we have decided to show the consumer to go a bit deeper in detail of each tweet like the id from where it was tweeted,the likes, the retweets, the dates and a few more. We demonstrate an even more vivid experience to the user by providing him a graph like tweets each day with respect to the dates and a consequesnt graph between the retweets and dates.

MODELING AND PREDICTION

Our model is trained using the Random Forest Classifier, which is a is a type of supervised machine learning algorithm based on ensemble learning. The f1 score of the model is 84.26 (84% accuracy). Thus finally we are predictig the tweet sentiment with our model and showing the user what does the market regard him as, whether his tweets are positive one or they do suffer an adverse effect.
